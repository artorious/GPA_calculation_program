# GPA Calculation Program
A simple program that computes a semester GPA &amp; cumulative GPA for a given student.

Programming features:

    * tuple assignment

Usage
=====
Run:
<pre>
$ <kbd>python3 semester_gpa.py</kbd>
</pre>
